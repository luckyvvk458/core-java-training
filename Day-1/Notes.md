# Day 1 - Introduction to Java

## Objective

Understand why Java was created, the problems it solved, and the features that made it one of the most popular programming languages.

---

## What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995.

Java follows the principle:

> Write Once, Run Anywhere (WORA)

This means that Java programs can run on any machine that has a Java Virtual Machine (JVM).

---

## Why Was Java Created?

Before Java, developers primarily used C and C++.

Although powerful, they had several challenges.

### Problems with C

* Manual memory management
* Use of pointers leading to memory corruption
* Platform dependency
* No object-oriented support

### Problems with C++

* Complex syntax
* Multiple inheritance issues
* Manual memory management
* Platform dependency

Java was introduced to simplify application development and provide better portability.

---

## Evolution of Java

### C Language

* Procedural Programming
* Functions and Structures

### C++

* Introduced Object-Oriented Programming
* Classes and Objects

### Java

* Pure Object-Oriented Approach
* Platform Independence
* Automatic Memory Management
* Rich Standard Library

---

## Features of Java

### Platform Independent

Java source code is compiled into bytecode.

The bytecode is executed by the JVM.

```text
Java Code
    ↓
Compiler
    ↓
Bytecode
    ↓
JVM
    ↓
Operating System
```

Because every platform has its own JVM, Java programs can run on multiple operating systems.

---

### Object-Oriented

Java is based on:

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

These concepts help in building reusable and maintainable applications.

---

### Robust

Java provides:

* Exception Handling
* Automatic Memory Management
* Strong Type Checking

These features make Java applications reliable.

---

### Secure

Java avoids many security issues by:

* Removing direct pointer manipulation
* Bytecode verification
* JVM security mechanisms

---

### Portable

Java programs behave consistently across different operating systems.

---

### Multithreaded

Java supports concurrent execution of tasks using threads.

Example:

* Downloading a file
* Playing music
* Updating UI

can happen simultaneously.

---

## Key Takeaways

* Java was created to overcome limitations of C and C++.
* JVM enables platform independence.
* Java supports Object-Oriented Programming.
* Automatic Garbage Collection simplifies memory management.
* Java follows the Write Once Run Anywhere principle.
